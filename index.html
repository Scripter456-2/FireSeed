FAQ

Common questions, quick answers.

What is FireSeed? ▾
FireSeed is a lightweight web browser built entirely in Python. It's designed to be fast, hackable, and open source.
What's the difference between Release and Snapshot? ▾
Release is the stable, tested version recommended for daily use. Snapshot includes the latest changes and experimental features — great for developers who want to stay on the cutting edge.
Is FireSeed free? ▾
Yes! FireSeed is 100% free and open source. No ads, no subscriptions, no catch. The source code is available on GitHub.
Does it auto-update? ▾
Yes. FireSeed has a built-in auto-update system that checks for new releases in the background. You can configure the update channel in settings.
What platforms are supported? ▾
Currently FireSeed is available for Windows. More platforms may be supported in the future as the project grows.
